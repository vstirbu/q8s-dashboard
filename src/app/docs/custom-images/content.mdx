---
title: Custom container images
---

You can provide a custom container image by passing it to the `DOCKER_IMAGE` environment variable:

```sh
docker run -p 8888:8888 \
  -e DOCKER_IMAGE="your-custom-image"
  -v /path/to/config:/home/jupyter/.kube/config  \
  -v /path/to/workspace:/workspace \
  --pull always \
  ghcr.io/torqs-project/q8s-devenv:main
```

You can create the custom image using `vstirbu/benchmark-deps` as base image. The base image already contains the CUDA dependencies for this cluster:

```Dockerfile
FROM vstirbu/q8s-cuda12
...

```
